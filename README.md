# Repositório de Projetos AWS

Este repositório contém projetos desenvolvidos com foco em **implementação e gestão de serviços na nuvem utilizando AWS**.  
Os projetos têm como objetivo aplicar soluções práticas para **redução de custos, escalabilidade, alta disponibilidade e eficiência operacional** em ambientes corporativos.

O repositório demonstra a capacidade de planejar, implementar e documentar soluções de infraestrutura na nuvem, utilizando os principais serviços da Amazon Web Services.

## Tecnologias e Serviços Utilizados

- **AWS EC2**: provisionamento de servidores virtuais para execução de aplicações  
- **AWS RDS**: gerenciamento de bancos de dados relacionais com alta disponibilidade  
- **AWS S3**: armazenamento de arquivos e conteúdos estáticos com durabilidade e baixo custo  
- **AWS CloudWatch**: monitoramento de recursos e performance  
- **AWS IAM e VPC**: segurança, controle de acesso e isolamento de rede  
- **Python**: scripts de automação e integração com serviços AWS  
- **Git e GitHub**: versionamento e organização dos projetos  
- **Diagramas e documentação**: representação visual da arquitetura e processos implementados

## Descrição dos Projetos

### [01 – Redução dos Custos em Farmácias com AWS](01-reducao_de_custos_com_aws/)
Projeto focado na criação de uma infraestrutura em nuvem escalável e eficiente, utilizando serviços AWS essenciais para operações corporativas.  
O projeto inclui:

- Provisionamento de servidores EC2 para hospedar aplicações web  
- Configuração de RDS para gerenciar banco de dados relacional com backups automáticos e alta disponibilidade  
- Armazenamento de arquivos estáticos e multimídia em S3 com políticas de durabilidade e otimização de custos  
- Monitoramento e logs de performance via CloudWatch  
- Documentação completa da arquitetura da solução  

Principais conceitos aplicados:
- Planejamento de infraestrutura em nuvem  
- Otimização de custos operacionais  
- Alta disponibilidade e escalabilidade  
- Segurança e controle de acesso em ambientes AWS
